# Email Marketing Trends Dashboard (Power BI)

## Overview
This project is an interactive Power BI dashboard that analyzes email marketing trends in India. It provides insights into user demographics, engagement patterns, and regional distribution.

---

## Key Features
- Interactive filters for Year, State, and City  
- Analysis by Age Group and Gender  
- Visualization of enrollment trends over time  
- City-wise and state-wise data distribution  
- Business vs Individual user comparison  

---

## Tools & Technologies
- Power BI (Data Visualization)  
- Data Cleaning & Transformation  
- DAX (Data Analysis Expressions)  

---

## Dashboard Insights
- Total IDs and enrolled cities overview  
- Monthly enrollment trends  
- State-wise distribution of users  
- Age group and gender analysis  
- Marital status insights
